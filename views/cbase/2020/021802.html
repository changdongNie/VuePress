      # 代码风格

      1. 语句以分号（;）结尾，语句可以写在多行代码上。
      2. 花括号（{}）把语句组合为块
      3. 单行注释(//),多行注释（/* 代码 */）
      4. c#区分大小写

      # 变量

      变量声明： 数据类型 变量名； 如：int i; i=20; int j=10;。

      1. 同名的局部变量不能在同一作用域声明两次 如：int x=10； int x=20; 会报错。
      2. 类级别声明变量看作为字段，方法中声明的为局部变量，避免字段和局部变量冲突，使用变量前需须初始化。
      3. 可类型推断var 关键字声明变量 如：var name="nie"; var age=23;var isRabbit=false 声明必须初始化，变量类型确定后，不能更改变量类型。
      4. 常量定义 const 如 const int a=3; 常量声明时必须初始化，指定值后不能修改，常量包含 static修饰符
      5. 只读字段定义 readonly，一般在构造函数中初始化且不能修改。

      # 变量作用域

      变量存在于方法块、for、while、if等语句中，作用域范围在花括号内（{ }）。

      using System;
namespace ClassSample
{
  class Program
  {
      //Main方法程序入口
      static void Main(string[] args)
      {
          Person person=new Person("nie");
          person.Write();
      }
     
  }
  public class Person{
      public  string name;
      public Person(string Name){
          this.name=Name;
      }
      //字段
      public int age=22;
       public void Write(){
           //变量
          var isRabbit=false;
           string name = "Rabbit"; 
          //仅在当前for作用域内可访问
          for(int i=9;i>=0;i--){
              Console.WriteLine($"数字{i}");
          }
          //while局部作用域中可访问字段age
          while(age>21){
              Console.WriteLine("年龄"+age);
              age--;
          }
          //字段和局部变量冲突
          if(!isRabbit){
              //this.name调用字段，name调用局部变量
              Console.WriteLine(this.name+"no兔子,兔子是："+name);
          }
      }
  }
}

      # 值类型

      概念： 直接存储值，值类型存储在（stack）中。

      .NET内置值类型表示的基本类型有：整型、浮点类型、bool类型、字符类型。

      # 整型

      名称 .NET类型 说明
      sbyte System.SByte 8位有符号的整数
      short System.Int16 16位有符号的整数
      int System.Int32 32位有符号的整数
      long System.Int64 64位有符号的整数
      byte System.Byte 8位无符号的整数
      ushort System.UInt16 16位无符号的整数
      uint System.UInt32 32位无符号的整数
      ulong System.UInt64 64位无符号的整数

      # 引用类型

      **概念：**存储对值的引用，引用类型存储在托管堆（managed heap）上。

      预定义引用类型：Object和string

      # Object类型

      **概念： ** c#中，所有内置类型和用户定义类型都派生自Object类。

      作用：

      1. 使用object引用绑定任何特定子类型的对象。如：值类型装箱拆箱。反射处理未知对象。
      2. 提供基本方法，包括Equals()、GetHashCode()、GetType()和ToString()。可通过重写替代object中的代码实现

      # string类型

      1. sring对象被分配在堆上，字符串是不可改变的

      2. '单引号为char类型,"双引号为string类型

      3. 字符串和char包含Unicode和十六进制数转义序列，反斜杠为转义字符（\\） ，@ 符号可以取消转义。$字符串插值格式

        public static void Main(){
    string s1="a";
    //s2指向 a
    string s2=s1;
    //改变s1指向 b
    s1="b";
    //s2指向 a
    Console.WriteLine(s2);
    //反斜杠需要转义
    string s1="c:\\proCsharp\\First.cs";
    string s2=@"c:\proCsharp\First.cs";
    //字符串插值格式
    Console.WrinteLine($"s1 is{s1}");
}

        # if语句

        1. if表达式中必须等于布尔值（Boolean），不能直接测试整数（从函数中返回的值），必须显式把返回值的整数转换为布尔值true或false。
        2. if、 else if 如果满足执行条件就会执行相应语句，如果都不满足将执行else 语句块。
        int i=3;
if(i==3){//满足条件输出
    Console.WriteLine("输出");
}else if(i==2){
     Console.WriteLine("不输出");
}else if(i<4){//满足条件输出
    Console.WriteLine("输出")
}
else{
     Console.WriteLine("不输出");
}

        # Switch语句

      1. switch ... case 中case值不能相同，case值必须是常量，参数匹配分支case值为true的执行。
      2. 表达式不等于任何case子语句的值，就执行default子语句
      3. break 跳出
      4. default 代表没有匹配项执行
      5. string 可以作为case值
      int integerA=3；
switch(integerA){
 case 1:
 Console.WriteLine("第一种情况");
 break;
 case 2:
 Console.WriteLine("第二种情况");
 break;
 case 3:
 case 4://执行integerA为值为3或4的情况
 Console.WriteLine("第三种情况");
 break;
 default:
 Console.WriteLine("第四种情况");
 break;
}

      # 命名空间

      1. 命名空间是一种逻辑组合，不是物理组合，不同文件夹下面可以存在相同的命名空间
      2. 通过using语句引用类型，如果两个命名空间包含同名的类型，必须完整的名称
      3. "::"字符可以制定别名
      using System;
using Introduction=Wrox.ProCSharp.Basics;
class Program{
static void Main(){
   Introduction::NamespaceExample NSEx=new Introduction::NamespaceExample();
   Console.WriteLine(NSEx.GetNamespace());
}
namespace Wrox.ProCSharp.Basics{
    class NamespaceExample{
        public string GetNamespace(){	
        return this.GetType().Namespace;
        }
    }
}
}
